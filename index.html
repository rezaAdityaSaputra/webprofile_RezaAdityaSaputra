Skill

Web development

Web development adalah proses pembuatan dan pemeliharaan situs web atau aplikasi web. Ini melibatkan sejumlah tindakan seperti merancang tampilan dan fungsionalitas situs, menulis kode pemrograman, mengatur database, dan mengelola server web.

UI/UX Design

UI/UX design adalah proses perancangan tampilan dan interaksi antarmuka pengguna pada situs web. Ini mencakup elemen-elemen seperti merancang layout dan grafis, serta memastikan pengalaman pengguna yang intuitif dalam penggunaan situs tersebut.

Mobile Development

Mobile development adalah proses menciptakan dan memelihara aplikasi mobile. Ini termasuk tindakan seperti merancang tampilan dan fungsionalitas aplikasi, menulis kode pemrograman, dan mengoptimalkan kinerja aplikasi untuk perangkat mobile.
